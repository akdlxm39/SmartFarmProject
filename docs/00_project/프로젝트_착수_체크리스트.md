# SmartFarmProject 프로젝트 착수 체크리스트

작성일: 2026-06-22
상태: 프로젝트 착수 단계

## 1. 현재 확정된 큰 방향
- 3인 협업 프로젝트
- 현실 구현과 시뮬레이션 구조를 분리해서 관리
- 현실 구현: 불량 즉시 분리 + 정상 컨베이어 이송
- 시뮬레이션: 3열 분류 + AGV/TurtleBot 물류 흐름 표현
- TurtleBot SLAM/자율주행은 MVP 포함
- Notion은 인증 준비 후 별도 생성
- GitHub 저장소는 생성 완료

## 2. 바로 시작할 1차 작업
### 공통
- WBS.xlsx 기준 팀원별 담당자 입력
- GitHub 저장소 공유 및 접근 확인
- 프로젝트 폴더 구조 합의

### 비전/임베디드
- 작물 3종과 불량 기준 확정
- 데이터셋 수집/라벨링 기준 작성
- 1번 카메라 3방향 촬영 입력 구조 설계
- 2번 RGB-D 카메라 위치/시야 확인

### 로봇/시뮬레이션/자율주행
- Dobot 수확/촬영/폐기/컨베이어 적재 시퀀스 작성
- RoboDK 시뮬레이션 3열 분류 구조 구체화
- TurtleBot SLAM 환경 세팅 및 기본 맵핑 테스트
- 정상 수거 상자/불량품 상자 위치 배치도 작성

### 서버/프론트/통합
- Django/MySQL 프로젝트 초기화 계획
- 수확 로그/판정 결과/장치 상태 DB 필드 초안
- Vue 대시보드 와이어프레임
- ROS-Django 연동은 후순위로 인터페이스만 먼저 정리

## 3. 오늘 정하면 좋은 것
1. 각 Participant 실제 이름 매핑
2. WBS 각 행 담당자
3. 실제 작물 3종
4. 불량 기준의 구체적 정의
5. TurtleBot SLAM 테스트 환경

## 4. 관리 원칙
- 결정사항은 `docs/작업_결정_메모.md`에 누적
- 진행상황은 `docs/진행_로그.md`에 누적
- 시뮬레이션/현실 구현 차이는 별도 문서로 계속 관리
- 큰 변경은 Git 커밋으로 남김
