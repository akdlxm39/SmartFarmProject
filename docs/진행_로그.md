### 핵심 결과
- 필수 산출물 축을 `주제 / 기술조사 / 아이디어선정 / 기술스택 / SW구조 / WBS / R&R`로 정리했다.
- 초기 기능 블록을 `비전AI / Dobot / 컨베이어 / TurtleBot / 웹관제` 5개로 정리했다.
- Dobot은 석션컵으로 작물을 수확한 뒤 1번 카메라 앞으로 이동하고, 작물을 **120도 간격 3방향**으로 촬영한다는 점을 확인했다.
- 이후 불량 작물은 별도 상자에 담고, 정상 작물만 컨베이어에 올려 2번 카메라가 벨트 위 흐름과 위치를 확인한다는 흐름을 정리했다.
- 분류 기준은 `작물 3종 분류 + 썩음 여부 판별`이며, 학습된 비전 모델을 사용한다.
- 논의 결과, 현재 확정 방향은 `1번 카메라=작물 종류 분류 + 불량 판별`, `2번 카메라=라인 흐름 확인용 보조 비전`이다.

### 막힌 점 / 판단 필요
