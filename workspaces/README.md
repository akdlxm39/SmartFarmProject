# 개인 작업 폴더

초기 개발 단계에서 팀원별 작업물을 임시로 올리는 공간입니다.

## 폴더 구조
- `지성/`
- `지웅/`
- `효진/`

## 운영 방식
1. 각자 담당 파트의 실험 코드, 초안, 테스트 파일을 본인 폴더에 먼저 올립니다.
2. 기능 뼈대가 잡히면 공통 구조로 통합합니다.
3. 통합 후에는 개인 폴더의 코드를 그대로 운영하지 않고, 공통 폴더로 이동한 버전을 기준으로 관리합니다.

## 이후 통합 후보 구조
- `vision/`: 카메라, AI 모델, 추론 파이프라인
- `robot/`: Dobot, TurtleBot, ROS 2 제어
- `backend/`: Django, DB, API
- `frontend/`: Vue, Chart.js 대시보드
- `simulation/`: RoboDK/Gazebo/RViz 등 시뮬레이션
- `docs/`: 설계/운영/회의 문서
